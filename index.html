Belajar SQL

SQL adalah bahasa standar untuk mengakses dan memanipulasi database.

Dalam tutorial ini, Anda akan mempelajari bagaimana menggunakan SQL untuk mengakses dan memanipulasi data dalam database seperti MySQL, PostgreSQL, SQL Server, Oracle, dan lainnya.

Apa itu SQL?

  • SQL adalah singkatan dari Structured Query Language
  • SQL memungkinkan Anda mengakses dan memanipulasi database
  • SQL menjadi standar ANSI (American National Standards Institute) pada tahun 1986
  • SQL dapat menjalankan query terhadap database
  • SQL dapat mengambil data dari database
  • SQL dapat memasukkan record dalam database
  • SQL dapat memperbarui record dalam database
  • SQL dapat menghapus record dari database
  • SQL dapat membuat database baru
  • SQL dapat membuat tabel baru dalam database

SQL di Website Anda

Untuk membuat website yang menampilkan data dari database, Anda akan membutuhkan:

  • Program database seperti MySQL, PostgreSQL, MS SQL Server
  • Bahasa scripting sisi server seperti PHP, Python, Node.js
  • SQL untuk mendapatkan data yang Anda inginkan
  • HTML/CSS untuk styling halaman

Database Apa yang Menggunakan SQL?

Ada banyak sistem database yang menggunakan SQL, antara lain:

  • MySQL - Database open source yang populer
  • PostgreSQL - Database open source yang powerful
  • Microsoft SQL Server - Database dari Microsoft
  • Oracle - Database enterprise yang powerful
  • SQLite - Database ringan untuk aplikasi mobile
  • MariaDB - Fork dari MySQL
